Defaults Configuration

This part of the tool allows you to determine the default values we use on various forms, mostly the blueprint browser screen for now. There are four tabs a the moment.
  • Main - This is the Blueprint Browser combos at the top. The input order type, outpur order type, and whether or not you choose to invent the blurprint
  • Manufacturing Defaults - Default values for the Manufacturing section of the Blueprint Browser.
  • Invention Defaults - Default values for the Invention section of the Blueprint Browser
  • Reaction Defaults - Default values for the Reactions section of the Blueprint Browser.
