Install Directions

  1. Download the zip file from the dropbox link above
  2. Extract the files
  3. Go into the folder and click the file named "setup". This is the installer.
  4. The installer may ask you to install some desktop runtime libraries, currently, .net 7. This is just Microsoft code that does not normally come with your base install of windows but the app needs it to run correctly. Install this dependency.
  5. After all dependencies have been installed, a second window will ask you to install the app. Since this is a beta release you will get a warning saying the publisher couldn't be verified. This is because I have not gotten a security certificate. This is ok for now. When I do the full release after beta testing is complete, the app will get a certificiate.
  6. Once the app is installed, it will auto-run. You can access it anytime by searching for 'EveHelper' in the start menu.
  7. Done. ENjoy the app.

What is Eve Helper?

Eve Helper is a companion app to Eve Online, developed by CCPGames. I created this app to replace a myriad of tools that used to be hosted online by various third party developers which have since gone dark.

Why a Windows app and not a website?

Websites cost money to maintain, apps deployed like this don't. I do this in my spare time for myself. I'm only deploying this because other people showed interest. In addition, if I take a break from Eve and decide not to udpate the app, you can update it yourself. No need for me. More on that later.

What does this app do?

This app currently has the the below tools.
  • Blueprint Browser - Blueprint Cost Calculator. Includes Research, Manufacturing, and Reactions
  • Planetary Interaction Guid - Gives a list of items you can produce on planets and shows you the build tree for how to make it
  • System FInder - Helps you find systems with matching criteria.
  • Loot Appraisal - Copy and paste items from Eve and I give you the market value
  • Market Browser - Allows you to lookup items on the market. Also captures price history.
  • Price History Utility - Allows you to download price history data and view it for items.
  • Abyss Tracker Utility - Allows you to track the profit you're making in the Abyss
  • Default Configuration - Allows you to configure defaults used elsewhere in the app.


How do I update the app myself?

The app is built to use the static data dump SQLite file that is provided by fuzzworks. This means that if you want to update the app to use the new data from a new expansion, all you have to do is go and download the new static data SQLite file from Fuzzworks website and then replace the current file the app is using. I will be adding a built-in method that allows you to do this without knowing where the apps files are located. This way you can download the SDE dump and then select the file from your downloads folder.
